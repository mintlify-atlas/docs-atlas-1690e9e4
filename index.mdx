Osintgram is a Python-based OSINT (Open Source Intelligence) tool that lets you run reconnaissance on any public Instagram account by username. It provides an interactive shell with tab-completion, or a single-command mode for scripting — extracting geolocation data, follower lists, contact information, hashtags, captions, and more.

<Warning>
  For educational purposes only. Do not use Osintgram on accounts without authorization. The contributors assume no responsibility for misuse of this tool. Avoid using your primary Instagram account when running Osintgram.
</Warning>

## What Osintgram can do

Osintgram targets a public Instagram account by nickname and lets you run any of its built-in intelligence commands in an interactive session or from the command line.

## How it works

<Steps>
  <Step title="Install and configure">
    Clone the repo, create a virtual environment, install dependencies, and add your Instagram credentials to `config/credentials.ini`.
  </Step>
  <Step title="Set a target">
    Run `python3 main.py <target_username>` to start the interactive shell pointed at any public Instagram account.
  </Step>
  <Step title="Run commands">
    Type a command like `info`, `followers`, or `addrs` at the prompt. Results print to the terminal and can be saved as `.txt` or `.json`.
  </Step>
  <Step title="Export results">
    Enable file output with `FILE=y` or JSON export with `JSON=y` at any time during a session to persist all subsequent command results.
  </Step>
</Steps>

<Note>
  Osintgram works only on **public profiles** or profiles you already follow. Private account data is not accessible.
</Note>
